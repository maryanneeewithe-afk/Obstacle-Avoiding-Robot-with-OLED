# Obstacle-Avoiding-Robot-with-OLED (Wall-E inspired)

A simple obstacle-avoiding robot that can detect objects from the front, left, and right using an ultrasonic sensor and a servo motor, and provide real-time status monitoring through an OLED display.


---------------------------------------------------------------------------------------------------------

Materials used: 

- 1x Arduino Uno
- 1x L298N motor driver
- 1x Ultrasonic sensor (HY-SRF05)
- 1x Servo motor (SG90)
- 1x Battery holder (4 slots)
- 4x Battery 18650 3.7v
- 1x 0.91 OLED 
- 4x car wheels
